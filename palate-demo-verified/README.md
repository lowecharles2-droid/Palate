# Palate Verified Demo

A dependency-free Node.js proof-of-concept for restaurant discovery near UC San Diego.

## What changed

- 50 source-checked restaurant listings
- Official website on every listing
- Verification source and last-checked date
- Transparent cuisine-fidelity and local-ownership scores
- Evidence-based dietary and menu tags
- No invented calorie, protein, allergy, or certification claims
- Collapsible restaurant sections to prevent endless scrolling

## Run locally

```bash
npm start
```

Open `http://localhost:3000`.

## Data policy

The restaurant data was checked on July 6, 2026 against current official university, shopping-center, and restaurant pages. Restaurant hours, menus, and operating status can change. The app links to the official website and verification source so users can confirm current details. Distances are approximate from central UC San Diego and are not live route distances.

The Palate authenticity score is an editorial cuisine-fidelity score, not an objective cultural judgment. It measures how closely a restaurant delivers the category it claims to offer. Local ownership is scored separately.
